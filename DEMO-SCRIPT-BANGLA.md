# ৫–৭ মিনিটের Demo Script

## ১. Introduction

“আমাদের project-এর নাম Anti-Kuddus Protocol। Junior segment-এর ছয়টি baseline mission আমরা একটি full-stack ecosystem-এর মধ্যে combine করেছি। Frontend React, backend Express এবং database হিসেবে SQLite ব্যবহার করেছি।”

## ২. Student Login

Roll `701` দিয়ে login করো। বলো:

“Roll number frontend-এ hardcode করা নেই। Server-এর private mapping-এর সঙ্গে verify হয়। Complaint table-এর সঙ্গে roll store হয় না।”

## ৩. Mission 1

Whistleblower page-এ complaint submit করো। Progress bar পরিবর্তন দেখাও। বলো:

“Warning count database-এর complaint total থেকে dynamically calculate হয়। তিনটি warning হলে impeachment state দেখায়।”

## ৪. Mission 2

Seat Planner-এ নতুন student add করো। Rows/columns change করো। বলো:

“Students height অনুযায়ী ascending sort হয়। Sorted list row-major order-এ front থেকে back grid-এ বসে।”

## ৫. Mission 3

Syllabus AI page-এ demo text summarize করো। বলো:

“API key server-side `.env` file-এ থাকে। Browser সরাসরি Gemini-কে call করে না। তাই API key frontend bundle-এ expose হয় না।”

## ৬. Mission 4

Payment বা stolen tiffin add করো। Chart update দেখাও। বলো:

“Ledger entry SQLite database-এ save হয়। Chart.js daily grouped cash এবং food data দেখায়।”

## ৭. Mission 5

Corridor select করে SOS পাঠাও। তারপর logout করে Captain login করো। Captain Board-এ alert দেখাও এবং resolve করো। বলো:

“Junior requirement অনুযায়ী captain dashboard active alert দেখায়। আমরা five-second polling ব্যবহার করেছি, কারণ real-time WebSocket senior requirement।”

## ৮. Mission 6

Fact-Checker-এ লিখো:

```text
Students must pay 2 Taka for washroom.
```

Matching official rule দেখাও। বলো:

“এটি semantic embedding না। Junior baseline অনুযায়ী relational rule table-এর উপর direct string এবং keyword matching।”

## ৯. Final

Submission Check page দেখাও। বলো:

“সব junior baseline complete। Advanced senior features ইচ্ছাকৃতভাবে scope-এর বাইরে রাখা হয়েছে, যাতে required features stable এবং explainable থাকে।”

## নতুন Competitive Feature Demo

মূল ছয়টি mission দেখানোর পরে সংক্ষেপে বলবে:

> “আমাদের system বাংলা ও ইংরেজি—দুই ভাষায় কাজ করে। User light, dark বা system theme ব্যবহার করতে পারে। Project-টি installable PWA, তাই mobile home screen-এ app-এর মতো install করা যায়। Network না থাকলে SOS device-এ queue হয় এবং internet ফিরে এলে automatically captain dashboard-এ sync হয়। জরুরি contact ও safety steps-ও offline পাওয়া যায়।”

Demo-তে language button, dark theme এবং SOS offline queue একবার দেখালে project-এর usability ও engineering value পরিষ্কার হবে।
